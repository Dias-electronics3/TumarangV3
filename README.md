# TumarangV3
Program ini merupakan pengembangan dari TumarangV2, di mana program dan hasil akhirnya memperbaiki dari kesalahan/kekurangan yang sudah dilakukan sebelumnya di TumarangV2 sebelumnya. Dengan Dokumentasi program yang lebih struktural dan desain PCB yang lebih baik. ini merupakan Final dari Tumarang yang sudah disempurnakan tanpa adanya bug.
